# GEMU: Gameboy EMUlator
GEMU is a GAMEBOY emulator written in golang.

## Installation
### Building from source
```
$ cd gemu
$ make
$ ls dist
gemu
```

## Usage
```
$ gemu -h
Usage of gemu:

gemu [-v] ROM
    -v    display version
```
